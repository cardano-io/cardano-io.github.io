Sidechains: Non-interactive proofs of proof-of-work

In this work we introduce and instantiate a new primitive for blockchain protocols called Noninteractive-Proofs-of-Proof-of-Work (NIPoPoWs) which can be adapted into existing proof-of-work-based cryptocurrencies.

Ouroboros: A Provably Secure Proof-of-Stake Blockchain Protocol

Ouroboros is the first blockchain protocol based on proof of stake with rigorous security guarantees. We establish security properties for the protocol comparable to those achieved by the bitcoin blockchain protocol.
